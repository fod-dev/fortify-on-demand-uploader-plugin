The Fortify On Demand API URL indicates to the plugin which Fortify On Demand system it should connect to.
The Fortify On Demand API URLs are:

https://api.emea.fortify.com [Africa, Middle East and Europe]
https://api.apac.fortify.com [Asia and Pacific]
https://api.ams.fortify.com [Americas]
https://api.sgp.fortify.com [Singapore]
https://api.trial.fortify.com [Trials]
https://api.sandbox.fortify.com [Development]
https://api.fed.fortifygov.com [US Federal]

Custom deployment URL rules:
Protocol: the system only supports http and https
Port number: any valid port number
DNS: a valid domain name or hostname

Important:This property cannot be blank, end with the character "/", contain spaces, or include DNS restricted characters
